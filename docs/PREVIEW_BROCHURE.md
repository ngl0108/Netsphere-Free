# NetSphere Free Intake Edition Brochure

## Positioning

NetSphere Free Intake Edition is an experience-first field edition for parser improvement and vendor output collection.

It is designed for:

- auto discovery trials
- auto topology trials
- connected NMS trials
- installed local collector trials without Docker
- sanitized raw-output contribution after value is proven
- multi-vendor parser improvement

It is not positioned as full enterprise GA.

## Core free edition value

- discover devices and approve them into inventory
- review topology, path trace, diagnosis, alerts, and observability
- experience connected NMS workflows before deciding to contribute data

## What users can do

- register and scan devices
- review inventory, topology, diagnosis, observability, and issue flows
- capture allowlisted read-only command output
- inspect the sanitized preview before submission
- upload masked contribution bundles

## What is intentionally blocked

- live config deployment
- destructive automation
- rollback execution
- privileged admin mutation flows
- external webhook delivery workflows

## Data handling promise

- raw originals are reviewed locally first
- sanitization is applied before upload
- only sanitized bundles are stored
- contribution uploads are explicit opt-in

## Safe initial collection scope

- facts and version
- inventory summary
- interface brief/status
- VLAN summary
- MAC table summary
- LLDP/CDP detail
- route summary
- OSPF/BGP summary
- EVPN/VXLAN summary

## Explicit exclusions

- full running-config
- AAA and user database outputs
- certificate/private-key content
- full firewall policy dumps
- NAT/session dump collections
- cloud credential material

## Ideal audience

- operators willing to test device discovery and visibility
- partners helping expand vendor parser quality
- early design partners validating device coverage
